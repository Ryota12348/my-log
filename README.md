# my-log

私の考え、学習、そして日々の生活を記録するためのオールインワンリポジトリです。

これは、アイデアを保存し、進捗状況を追跡し、時間の経過とともに成長を振り返るためのスペースです。

---

## 📅 journal
日々のエントリ、イベント、そしてちょっとした反省。

## 📚 study
学習ログ、メモ、要約、そしてLaTeXの練習。

## 🧠 ideas
思いつき、コンセプト、そして初期段階の計画。

## 🛠 projects
小さな実験と個人的な創作。

---

> [!NOTE]
> 完璧よりも進歩。
> たった一行でも記録する価値はある。

---

## 🧭 原則

- 短くても、何かを書きましょう。
- 構造を使いましょう（見出しをつけると効果的です）。
- 定期的に見直し、整理しましょう。
