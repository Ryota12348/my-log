## 12. LaTeX（数式）

インライン：\
オイラーの公式は $e^{i\pi} + 1 = 0$ です。

ブロック数式：

$$
\frac{d}{dx} x^2 = 2x
$$

よく使う例：

-   $\sqrt{x}$
-   $\sum_{i=1}^n$
-   $\int_a^b$
-   $\lim_{x \to 0}$
-   $a^2 + b^2 = c^2$

## 13. Mermaid（図）

### フローチャート

``` mermaid
flowchart TD
    A[Start] --> B{条件}
    B -->|Yes| C[処理1]
    B -->|No| D[処理2]
```

### Gitの流れ

``` mermaid
flowchart LR
    A[編集] --> B[git add]
    B --> C[git commit]
    C --> D[git push]
```

### ガントチャート

``` mermaid
gantt
    title 勉強計画
    dateFormat  YYYY-MM-DD
    section 数学
    問題集 :done, 2026-02-01, 5d
    section 英語
    単語暗記 :active, 2026-02-10, 7d
```

### シーケンス図

``` mermaid
sequenceDiagram
    User->>GitHub: push
    GitHub-->>User: OK
```
